# AutoTrek Visual Reboot v3 — Brand Lock (PREMIUM / BOLD / TRUSTWORTHY)

**Directive:** lock the visual system to three non-negotiable brand adjectives only:
1. **PREMIUM**
2. **BOLD**
3. **TRUSTWORTHY**

Everything below is constrained to those three signals. If a design choice does not strengthen at least one of them, remove it.

---

## 1) Brand Lock Translation (Design Intent)

### PREMIUM
- Feels expensive through restraint, precision, and material realism (not ornament).
- Elevated contrast, disciplined spacing, editorial composition, top-tier photography.
- Minimal visual clutter; each component looks engineered.

### BOLD
- Immediate visual command: strong hierarchy, decisive typography, confident scale.
- Clear focal points per section; no timid layouts.
- High-impact CTAs and headlines that read in under 2 seconds.

### TRUSTWORTHY
- Radical clarity: transparent process, legible information, real people and real work.
- Predictable, consistent interface behavior.
- Authentic imagery and proof elements over marketing fluff.

---

## 2) Tightened Palette System (Brand-Locked)

## Core Colors
- **Midnight Black** `#0A0D12` (foundation / authority)
- **Graphite** `#161C24` (surface depth)
- **Porcelain White** `#F4F7FB` (clarity / readability)
- **Steel Gray** `#9AA6B5` (supporting text / dividers)
- **Command Orange** `#F65A1E` (bold conversion signal)

## Controlled Accent
- **Verification Blue** `#1E9BFF` (trust cues: process, data, focus states)

### Usage Ratio
- 70% Midnight/Graphite
- 22% Porcelain/Steel
- 8% Orange/Blue accents combined

### Lock Rules
- **Orange = action only** (book/call/primary conversion).
- **Blue = verification only** (steps, trust metrics, focus rings, technical data).
- Never use more than one accent as primary in a single component.
- No decorative gradients as primary backgrounds; use solid confidence first.

---

## 3) Typography Lock (Confidence + Legibility)

## Primary Stack
- **Display:** `Teko` 600/700 (BOLD)
- **Body/UI:** `Inter` 400/500/600 (TRUSTWORTHY clarity)

## Type Behavior
- Headlines: uppercase for hero/section openers, tight line-height.
- Body: sentence case, high readability.
- Numeric/stat content: Inter 600 with tabular alignment where possible.

## Scale (Desktop)
- H1: 64 / 0.92 / 700
- H2: 44 / 1.00 / 700
- H3: 30 / 1.10 / 700
- Lead: 20 / 1.45 / 500
- Body: 17 / 1.60 / 400
- Small UI: 14 / 1.45 / 500

## Non-Negotiables
- Never use thin display weights.
- Never center long-form body copy blocks.
- Maintain high contrast (minimum WCAG AA).

---

## 4) Imagery Lock (Authenticity + Authority)

## Image Style
- Real technicians, real bays, real tools, real service moments.
- Crisp, directional lighting on metal/surfaces.
- Color grade: cool-neutral base with natural skin tone retention.

## Priority Subjects
1. Technician executing skilled work (trust through competence)
2. Diagnostic instruments/readouts (trust through proof)
3. Precision detail shots (premium craft)
4. Advisor-customer explanation/handoff (human trust)

## Composition
- Hero: asymmetric framing with clear text-safe zone.
- Shots should show depth (foreground/mid/background) to feel cinematic and premium.
- Avoid over-staging and stock-photo body language.

## Prohibited
- Generic smiling stock photography
- Fake lens flares / heavy HDR gimmicks
- Overly saturated “salesy” color treatment

---

## 5) Texture + Surface Language

Use texture to imply engineered quality, not noise.

- **Micro-noise layer:** 2–4% opacity on dark shells (premium tactility)
- **Gridline overlay:** subtle technical lines at 5–8% opacity (trust via precision)
- **Panel edges:** crisp 1px borders in low-contrast steel/blue tones
- **Corner treatment:** restrained cut-corner motif (8px) on key cards/buttons only

No heavy grunge textures. No distressed effects. No decorative clutter.

---

## 6) Layout & Composition Lock

- One dominant message per viewport.
- One focal action per section.
- Strong left-to-right hierarchy; fast scan in 3 beats: headline → proof → CTA.
- Generous whitespace to signal premium confidence.
- Section rhythm alternates dark authority blocks and light clarity blocks.

## Grid
- Desktop: 12 columns, max width 1280px
- Mobile: 4 columns, high tap comfort and compressed copy density

---

## 7) UI Component Guidance by Adjective

### PREMIUM Components
- Elevated card shells with disciplined spacing and shadow restraint.
- Clean iconography (stroke-based, no novelty fills).
- High-quality media containers with consistent radius and crop logic.

### BOLD Components
- Primary CTA buttons with strong fill (Command Orange), medium-large hit area.
- Headlines that occupy meaningful visual mass.
- Trust stats in strong numeric blocks, not tiny metadata rows.

### TRUSTWORTHY Components
- Process timeline (clear steps, clear outcomes).
- Proof strips: certifications, warranty messaging, review score, years active.
- Form states that are explicit and calm: focus, validation, error, success.

---

## 8) Section Application (Homepage)

1. **Hero:** Bold headline + concise proof + primary CTA (Orange)
2. **Trust Strip:** certifications/warranty/rating/years in one fast-scanning row
3. **Services:** clear category cards with confidence language, no fluff copy
4. **Process:** 4-step “what happens to your vehicle” transparency block
5. **Proof Gallery:** real work + short technical callouts
6. **Reviews:** fewer, stronger testimonials with rating context
7. **Final CTA:** single conversion ask, zero competing actions

---

## 9) Brand QA Checklist (Hard Gate)

A design passes only if all are true:
- **PREMIUM:** Does it look intentionally restrained and high-quality?
- **BOLD:** Is the hierarchy decisive and impossible to miss?
- **TRUSTWORTHY:** Is the process/proof clear without interpretation?

If any answer is “no,” revise before ship.

---

## 10) Final Non-Negotiables

- No generic local-shop visual clichés.
- No aesthetic drift into playful, soft, or trendy-for-trend’s-sake styles.
- No ambiguity in CTA intent.
- No fake imagery.
- Every page must feel: **PREMIUM. BOLD. TRUSTWORTHY.**
