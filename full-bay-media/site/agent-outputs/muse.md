# AutoTrek Website — Visual Hierarchy Review + Design Direction Brief

## 1) Current State (Quick Audit)

### What’s working now
- Strong hero photography and clear core offer (“Reliable Auto Repair in Littleton, CO”).
- Repeated primary CTA (“Schedule an Appointment”) improves conversion opportunities.
- Helpful local SEO and trust content exists (service areas, FAQs, reviews, warranty mention).
- Consistent card style and clean overall layout.

### What’s weakening hierarchy/conversion
- **Too many near-equal sections** after hero (info cards, about, services, AI/SEO panel, trust, gallery, reviews, FAQ, final CTA). Visual weight is spread too evenly.
- **Primary trust proof is diluted**: key confidence triggers (warranty, years in business, review quality/volume, certifications) are present but not grouped into a dominant “proof strip.”
- **Section order buries conversion intent**: users likely want “Can you fix my issue?” + “Can I trust you?” + “How do I book?” in that sequence.
- **Typography contrast is modest** (body and subheads often close in perceived weight), so scanning speed can improve.
- The “Google + AI Search Visibility Method” block is internally useful but **customer-irrelevant above the fold** for first-time visitors.

---

## 2) Recommended Visual Hierarchy Strategy

### Priority ladder (what should stand out most)
1. **Book now / call now** (single dominant action)
2. **Core promise** (trusted local repair + who it’s for)
3. **Trust markers** (warranty, years, certifications, reviews)
4. **Service pathways** (most common jobs)
5. **Supporting reassurance** (about, process, FAQs, gallery)

### Page scanning model
- Treat the homepage as a **decision funnel**, not a brochure.
- Each section should answer one question quickly:
  1) What do you do?
  2) Can I trust you?
  3) Do you handle my problem?
  4) How do I book?

---

## 3) Typography Scale (Desktop + Mobile)

Use a tighter, intentional scale to separate headline levels and increase readability.

### Suggested type tokens
- **Display / H1**: 48/56 (desktop), 36/42 (mobile), weight 700
- **H2**: 36/44 (desktop), 30/36 (mobile), weight 700
- **H3**: 24/32 (desktop), 22/30 (mobile), weight 650–700
- **Lead paragraph**: 20/30 (desktop), 18/28 (mobile), weight 400–500
- **Body**: 17/28 (desktop), 16/26 (mobile)
- **Small/supporting**: 14/22
- **Overline/eyebrow**: 12/18, uppercase/semibold with tracking

### Usage rules
- Keep **one H1** only (already correct).
- Every major section starts with H2.
- Use lead copy directly under H1 and key H2 blocks.
- Keep paragraph widths ~60–75 characters for easy scan.

---

## 4) Spacing Rhythm (8px system)

Adopt a consistent vertical rhythm to avoid “same-density” sections.

### Spacing tokens
- Micro: 8
- Small: 16
- Medium: 24
- Large: 40
- XL section padding: 64
- XXL section padding (hero/CTA): 88–104

### Application pattern
- Section top/bottom: **64–88px desktop**, **48–64px mobile**
- Heading → subcopy: 12–16px
- Subcopy → content grid: 24–32px
- Card internal padding: 20–24px
- Card grid gaps: 16–24px

### Intentional contrast
Alternate section density:
- Dense (trust strip) → breathable (services) → dense (process/trust) → breathable (FAQ/CTA)

---

## 5) Section Order Improvements (Recommended)

### Proposed homepage order
1. **Sticky top bar / nav** with click-to-call CTA
2. **Hero** (promise + immediate CTA + secondary “View services”)
3. **Trust proof strip (new)**: 4–5 hard signals in one row
   - 3yr/36k warranty
   - Serving since 2009
   - ASE-certified technicians
   - Google rating + review count
   - European + fleet specialization
4. **Top services (problem-first cards)**
5. **Why choose AutoTrek** (short bullets + process)
6. **Reviews preview** (2–3 testimonial snippets + “Read all reviews”)
7. **About / shop photos** (merged, shorter)
8. **Service areas** (compact chips/map link)
9. **FAQ**
10. **Final high-contrast CTA band**

### What to move/reframe
- Move “Google + AI Search Visibility Method” off homepage (or to internal page/footer).
- Merge Address + Hours into a more compact “Visit & Contact” module near hero or above footer.
- Put reviews earlier (before long-form about content).

---

## 6) Trust Marker System (Make Credibility Immediate)

### Above-the-fold trust markers
- Row of icon + label chips under hero copy:
  - “ASE-Certified”
  - “3yr / 36k Warranty”
  - “Family-Owned Since 2009”
  - “European + Fleet Specialists”

### Mid-page trust module
- “Why drivers choose us” should include:
  - Short process (Inspect → Explain → Approve → Repair)
  - Warranty specifics
  - Parts quality language (OEM/quality equivalent)
  - Communication promise (text/call updates)

### Social proof enhancement
- Add review stats (rating + count) near first CTA.
- Include 2–3 real quote snippets with names/initials and vehicle type.
- Add partner/certification logos if available (ASE, NAPA, etc.).

---

## 7) Conversion/UI Direction

- Keep one primary CTA color (red) for booking; reserve green for secondary trust/accent actions.
- Ensure every major section ends with an action:
  - Call now
  - View services
  - Read reviews
- Mobile sticky CTA is good; add “Open now/Closed now” status beside phone for urgency/context.
- Standardize button hierarchy:
  - Primary: solid red
  - Secondary: outline dark/white
  - Tertiary: text link with arrow

---

## 8) Quick Win Implementation Plan

### Phase 1 (fast, high impact)
- Reorder sections to funnel logic.
- Add trust proof strip directly below hero.
- Move reviews section above About.
- Tighten heading/body scale and spacing tokens.

### Phase 2
- Add testimonial cards with rating snippets.
- Merge About + Inside AutoTrek into one concise “Local shop, real team” block.
- Compress address/hours into a compact contact module.

### Phase 3
- Add process section + certification logos.
- Expand service cards to “symptom/problem-first” language for faster self-selection.

---

## 9) Net Direction

The site already has the right ingredients. The next step is **hierarchy discipline**: fewer equal-weight sections, stronger trust proof early, and a clearer conversion path. Done right, the homepage should feel like: **Trust established in 5 seconds, service fit confirmed in 20 seconds, booking action obvious throughout.**
