## Services Overview Page
### Sections
1. Hero (service promise + CTA)
2. Who this is for (independent shops)
3. 4 core services (cards)
4. How the services work together
5. Process (audit → build → launch → grow)
6. FAQ
7. CTA (book audit)

---

## Individual Service Page Template
Use this structure for each service page:

1. **Hero**
   - Headline
   - 1-2 sentence value proposition
   - Primary CTA

2. **Problems We Solve**
   - 3-5 pain points this service fixes

3. **What’s Included**
   - Clear deliverables list

4. **How We Execute**
   - 3-5 step implementation process

5. **What Success Looks Like**
   - Practical outcomes / KPIs

6. **FAQ**
   - Timeline
   - What client needs to provide
   - Ongoing support

7. **CTA**
   - Book Audit / Start Project

## Service 1: Website Design & Build
### Suggested H1
Websites That Turn Local Drivers into Booked Jobs

- Outdated look hurts trust

- Poor mobile experience

## Service 2: Local SEO Optimization
### Suggested H1
Get Found First by Local Drivers

### Problems
- Low local rankings
- Weak service/location relevance
- Inconsistent metadata/content

### Includes
- On-page local SEO optimization
- Service/location keyword mapping
- Metadata + internal link cleanup
- Foundational technical SEO checks

### Success Metrics

- More calls from search

## Service 3: GBP Optimization & Management

## Service 4: Automated Review Funnels

Generate More 5-Star Reviews—Automatically
