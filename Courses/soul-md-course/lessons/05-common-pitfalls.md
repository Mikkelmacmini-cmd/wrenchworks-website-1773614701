# Lesson 5: Common Pitfalls

## Pitfall 1: Too abstract
Bad: “Be excellent.”
Fix: Write visible behavior.

## Pitfall 2: Contradictions
Bad: “Always concise” and “Always detailed.”
Fix: Add conditions for each.

## Pitfall 3: Missing escalation
Fix: Define when to ask, when to act, when to stop.

## Pitfall 4: No examples
Fix: Add 3 before/after snippets.

## Pitfall 5: No update loop
Fix: Review weekly or after incidents.
