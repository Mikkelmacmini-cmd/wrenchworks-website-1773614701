Exercises (short, for practice)

1) Spot the tacit knowledge
Read these items and decide which belong in MEMORY.md and why:
- "The server IP is 10.0.0.5" (no)
- "We always tag releases with semantic versioning" (yes)
- "Alice prefers async messages in the morning" (yes)

2) Write a rule from a story
Think of a time something went wrong. Write a one-line rule for MEMORY.md that would prevent it.

3) Merge two entries
Given:
- "Run full tests before deploy"
- "Deploy only after code review and tests"
Create a single clear rule and mark which old entries are superseded.

4) Create an entry using the template (see template.md)
Fill in the fields and save it.

5) Review practice
Take three entries from older notes and decide: keep, edit, or supersede. Write a one-line reason for your choice.
