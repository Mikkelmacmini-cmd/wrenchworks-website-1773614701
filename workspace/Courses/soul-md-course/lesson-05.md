Lesson 5 — Make it useful: examples and links

Goal: Add examples and links so people can act quickly.

Examples
- Show one short example of how to use or contribute to the project.
- Example: a one-command install or a one-paragraph workflow.

Links
- Link to key places: README, CONTRIBUTING, issue tracker, chat.
- Keep link labels clear: "Chat (Slack)", "Issues".

Exercise
- Add one example command and three helpful links for your project.
