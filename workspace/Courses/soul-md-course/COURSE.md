SOUL.md Course — Write a Friendly Project SOUL

Course goal
- Teach how to write a clear, useful SOUL.md file for a project.
- Language: simple, sixth-grade level.
- Deliverables: short lessons, templates, exercises with answers, and a one-page cheat sheet.

Course structure
1. Lesson 1 — What is SOUL.md?
2. Lesson 2 — The important sections
3. Lesson 3 — Writing the mission and voice
4. Lesson 4 — Rules and boundaries
5. Lesson 5 — Make it useful: examples and links
6. Lesson 6 — Review and publish

How to use this course
- Read each lesson file in order.
- Try the short exercise at the end of each lesson.
- Check your answers in the answers file.
- Use the template to make your own SOUL.md.
- Keep the cheat sheet for quick help.
