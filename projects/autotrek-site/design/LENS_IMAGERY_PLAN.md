# AutoTrek Service Center — LENS Imagery Plan

## Objective
Build visual trust fast for a clean, premium local auto service site: real shop credibility first, stock support second.

---

## 1) Required Photo Types by Website Section

## Homepage (Above the Fold)
- **Primary hero image (required):** Wide shot of the actual AutoTrek storefront/service bays with clear signage.
- **Secondary option:** Friendly advisor + technician near a vehicle in-bay.
- **Support detail:** Clean waiting area or front desk (for premium feel).
- **Avoid:** Generic stock mechanic closeups as hero.

## Homepage (Trust Strip)
- NAPA affiliation visual (badge lockup used per brand guidelines).
- Warranty visual card/graphic: **36 months / 36,000 miles**.
- 2–3 “real shop” thumbnails (team, bays, equipment).

## Services Overview Page
- Service-category headers (real photos preferred):
  - AC Service & Repair
  - Check Engine / Diagnostics
  - EV & Hybrid Repair
  - Brakes, Tires, Alignment, Suspension
- Use one strong image per service block + consistent crop ratio.

## Individual Service Pages
- **Top image:** Relevant real in-shop photo tied to that service.
- **In-content support:** 1–2 close detail shots (hands/tools/parts) that feel authentic.
- **Schema/relevance support:** Include vehicle/service-context visuals (scanner on dash for diagnostics, lift + brake job for brakes, etc.).

## About Page
- Team portrait (owner/lead tech + team, candid but professional).
- “At work” documentary shots in service bays.
- Exterior + interior sequence to reinforce physical location.

## Reviews / Social Proof Section
- Branded background texture or subtle shop image.
- Optional: Screenshot snippets only if readable and permission-safe.
- Keep imagery understated; testimonials should remain primary.

## Contact / Location Page
- Exterior daytime photo for “what to look for when arriving.”
- Parking/entrance angle.
- Optional map-adjacent photo tile showing nearby recognizable context.

## Local SEO Landing Sections (Littleton + nearby cities)
- Reuse core shop images with city-specific copy overlays only.
- Add 1–2 local-environment shots (roads/neighborhood context) where available.
- Do not use misleading “different shop” photos per city.

## 3) Icon Style Guidance

## Style System
- **Look:** Clean, geometric line icons with slight rounded corners.
- **Tone:** Professional/technical, not cartoonish.
- **Stroke:** 1.75–2.25px equivalent at 24px base.
- **Corner radius:** medium (harmonize with brand’s premium feel).
- **Fill:** Primarily outline; limited filled accents for hover/active states.

## Color Rules (brand-aligned)
- Primary icon color: deep neutral/charcoal for readability.
- Accent usage: red or green highlights only for key states, not full icon sets.
- White icons only on dark/color backgrounds with WCAG-safe contrast.

## Usage Rules
- Keep one consistent library style across all pages.
- Service icons should map to top services (AC, brakes, diagnostics, EV/hybrid, tires, alignment, suspension, transmission, oil/fluids).
- Avoid mixing duotone, skeuomorphic, and flat icon families.
- Size set recommendation: 20/24/32px with consistent optical alignment.

## Suggested Sources
- Tabler Icons (outline consistency, commercial-friendly)
- Lucide (lightweight modern strokes)
- Phosphor (if rounded variant needed; lock to one weight)

## 4) Placeholder + Source Recommendations

## Priority Order for Visual Assets
1. **Real AutoTrek photography** (highest trust + conversion impact)
2. **Local custom shoot placeholders** (temporary but realistic framing)
3. **Premium stock support** (only where real shots unavailable)

## Placeholder Strategy (during build)
- Use section-labeled placeholders with exact intended aspect ratio:
  - Hero: 16:9 (e.g., 1920x1080)
  - Service cards: 4:3 (e.g., 1200x900)
  - Team/about: 3:2
  - Thumbnails: 1:1
- File naming convention:
  - `hero-storefront-autotrek-v1.jpg`
  - `service-brakes-inbay-placeholder.jpg`
  - `about-team-group-placeholder.jpg`

## Stock Source Recommendations (if needed)
- Primary: Adobe Stock, iStock (higher quality, cleaner licensing)
- Secondary: Pexels/Unsplash (only for non-core decorative backgrounds)
- Selection criteria:
  - Realistic US shop environments
  - Modern vehicles and tools
  - No obvious fake staging or mismatched uniforms

## Licensing & Compliance
- Track every image source/license in a simple asset sheet.
- Avoid using competitor-branded shops/vehicles in any visible way.
- Confirm rights for any recognizable faces (model releases if stock).

---

## 5) Image Production Specs (Web)
- Formats: WebP primary, JPEG fallback.
- Quality targets: 70–82 (balanced for speed/clarity).
- Responsive set per key image: 640, 960, 1280, 1600, 1920 widths.
- Max hero weight target: ~250–350 KB optimized.
- Include descriptive alt text tied to service + location where relevant.
